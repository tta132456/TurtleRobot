
# Tower Stacking Robot

Created to compete in TURTLE Robotics' tower stacking competition. The competition featured a bracket format in which each game was played between two teams. Each team piloted their robots and attempted to move blocks.

from a shared space to their team's safe place. Once in the safe space, the team was permitted to stack blocks for additional points.

## Skills and Tools Used

* CAD Design (SolidWorks)

* 3D Printing (Bambu Labs Slicer and Printers)

* ESP32 Microcontroller

* Arduino Programming

* Workshop Tools (fasteners, drill bits, etc.)

* Wiring/Breadboarding

## Features

* Gripper: Utilizes a servo that is attached to arms. The arms rest on railings, such that the width between the arms can be adjusted to grip blocks. The servo itself resides in a 3D printed cage.

* Pulley: Has railings such that the servo cage can run up and down, thus allowing blocks to stack. An electrically controlled motor causes a length of fishing line to spool or unspool around a 3D printed pulley wheel.

* Mecanum Wheels: Enabled rotation of the robot to complete turns based on adjustment of each individual wheel speed.

## Parts

* 1 Servo

* 5 Electrical Motors

* 2 Motor Drivers

* 1 ESP32
